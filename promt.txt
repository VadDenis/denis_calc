# PocketBase JS Hooks Prompt Template

## Технический контекст
- Проект: Backend на PocketBase
- Язык: JavaScript (Goja runtime)
- Расположение: /pb_hooks/*.pb.js

## Ключевые ограничения JS кода на серверной части

### Runtime
- ✘ Нет async/await
- ✘ Нет npm пакетов
- ✘ Ограниченный ES6+
- ✓ Только синхронные операции
- ✓ Goja вместо V8/Node.js

### Безопасность
- ✘ Нет доступа к файловой системе
- ✘ Нет системных вызовов
- ✓ Ограниченный сетевой доступ
- ✓ Изолированное окружение

### API
- ✓ $app.dao() для БД операций
- ✓ Базовые HTTP операции
- ✘ Нет import/require
- ✘ Нет прямых HTTP клиентов

### Отладка
- ✓ console.log в stdout
- ✘ Нет source maps
- ✘ Ограниченный дебаггинг

## Рекомендации по коду

### Написание кода
- всегда пиши полный код, без // ... остальные поля ...

### стили и рекативность
- picocss
- alpine.js
